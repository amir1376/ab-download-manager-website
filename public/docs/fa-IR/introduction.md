# معرفی

به مستندات **AB Download Manager** خوش آمدید!

AB Download Manager یک دانلود منیجر رایگان، سریع و متن باز است که برای دانلودی روان، منظم و بسیار پرسرعت طراحی شده است.

## ویژگی‌های کلیدی

- **دانلود سریع**: با استفاده از اتصال های چند نخی، سرعت دانلود را تا ۵۰۰٪ افزایش می دهد.
- **مدیریت صف**: با ساخت صف های دانلود سفارشی، چندین فایل را به صورت پشت سر هم دانلود کنید.
- **رابط کاربری مدرن**: از یک رابط کاربری زیبا و امروزی با پشتیبانی داخلی از حالت تیره لذت ببرید.
- **یکپارچگی با مرورگر**: با استفاده از افزونه ها، لینک ها را از مرورگرهای محبوب (Chrome، Firefox، Edge، Opera و ...) به صورت خودکار دریافت می کند.
- **محدودکننده سرعت**: میزان پهنای باند مصرفی برنامه را کنترل کنید تا شبکه شما کند نشود.
