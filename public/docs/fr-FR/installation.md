## Installation

Vous pouvez installer AB Download Manager en utilisant les méthodes suivantes

### Téléchargement direct

- [Site officiel](https://abdownloadmanager.com/#download) : Téléchargez la dernière version stable directement. C'est l'option la plus simple pour la plupart des utilisateurs.
- [GitHub Releases](https://github.com/amir1376/ab-download-manager/releases/latest) : Contient les installateurs et l'historique complet des versions.

### Script d'installation

#### Linux (Recommandé)
```bash
curl -fsSL https://raw.githubusercontent.com/amir1376/ab-download-manager/master/scripts/install.sh | bash
```

### Gestionnaire de paquets

#### Windows

##### Winget

```bash
winget install amir1376.ABDownloadManager
```

##### Scoop :

```bash
scoop install extras/abdownloadmanager
```

#### Homebrew (pour macOS & Linux)

```bash
brew tap amir1376/tap && brew install --cask ab-download-manager
```
