# Introduction

Bienvenue dans la documentation d'**AB Download Manager** !

AB Download Manager est un gestionnaire de téléchargements gratuit, rapide et open-source conçu pour rendre le téléchargement de fichiers fluide, organisé et ultra rapide.

## Fonctionnalités clés

- **Téléchargements rapides** : Accélère vos téléchargements jusqu'à 500% en utilisant des connexions multi-threads.
- **Gestion de file d'attente** : Téléchargez plusieurs fichiers séquentiellement en créant des files d'attente de téléchargement personnalisées.
- **Interface élégante** : Profitez d'une interface utilisateur moderne et élégante avec prise en charge du mode sombre intégré.
- **Intégration avec le navigateur** : Capture automatiquement les liens depuis les navigateurs populaires (Chrome, Firefox, Edge, Opera, etc.) via des extensions.
- **Limiteur de vitesse** : Contrôlez la quantité de bande passante consommée par l'application pour éviter de ralentir votre réseau.
