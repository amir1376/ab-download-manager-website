# Intégration avec le navigateur

L'intégration d'AB Download Manager avec votre navigateur permet la capture automatique des liens de téléchargement, offrant une expérience de téléchargement fluide.

## Navigateurs pris en charge

Nous prenons en charge les navigateurs web modernes, notamment :
- **Google Chrome**
- **Mozilla Firefox**
- **Microsoft Edge**
- **Brave Browser**
- **Opera**
- **Vivaldi**

## Étapes d'installation

1. Installez l'extension de navigateur correspondant à votre navigateur web depuis le store officiel ou via les liens dans la fenêtre de téléchargement.
2. Assurez-vous que l'application de bureau AB Download Manager est en cours d'exécution en arrière-plan.
3. Une fois activée, les liens cliqués dans le navigateur seront automatiquement redirigés directement vers AB Download Manager.
