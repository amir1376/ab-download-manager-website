## Instalación

Puedes instalar AB Download Manager de las siguientes formas

### Descarga Directa

- [Sitio Web Oficial](https://abdownloadmanager.com/#download): Descarga la última versión estable directamente. Esta es la opción más simple para la mayoría de usuarios.
- [Lanzamientos de GitHub](https://github.com/amir1376/ab-download-manager/releases/latest): Contiene instaladores e historial completo de lanzamientos.

### Script de Instalación

#### Linux (Recomendado)
```bash
curl -fsSL https://raw.githubusercontent.com/amir1376/ab-download-manager/master/scripts/install.sh | bash
```

### Gestor de Paquetes

#### Windows

##### Winget

```bash
winget install amir1376.ABDownloadManager
```

##### Scoop:

```bash
scoop install extras/abdownloadmanager
```

#### Homebrew (para macOS y Linux)

```bash
brew tap amir1376/tap && brew install --cask ab-download-manager
```
