## التثبيت

يمكنك تثبيت برنامج مدير التنزيل AB باستخدام الطرق التالية:

### التنزيل المباشر

- [الموقع الرسمي](https://abdownloadmanager.com/#download): قم بتنزيل أحدث إصدار مستقر مباشرةً. هذا هو الخيار الأبسط لمعظم المستخدمين.
- [إصدارات GitHub](https://github.com/amir1376/ab-download-manager/releases/latest): يحتوي على ملفات التثبيت وسجل الإصدارات الكامل.

### سكربت التثبيت

#### لينكس (موصى به)
```bash
curl -fsSL https://raw.githubusercontent.com/amir1376/ab-download-manager/master/scripts/install.sh | bash
```

### مدير الحزم

#### ويندوز

##### Winget

```bash
winget install amir1376.ABDownloadManager
```

##### Scoop:

```bash
scoop install extras/abdownloadmanager
```

#### Homebrew (لـ macOS ولينكس)

```bash
brew tap amir1376/tap && brew install --cask ab-download-manager
```
