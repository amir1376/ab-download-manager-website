# Integración del Navegador

Integrar AB Download Manager con tu navegador permite la captura automática de enlaces de descarga, proporcionando una experiencia de descarga fluida.

## Navegadores Soportados

Soportamos navegadores web modernos, incluyendo:
- **Google Chrome**
- **Mozilla Firefox**
- **Microsoft Edge**
- **Brave Browser**
- **Opera**
- **Vivaldi**

## Pasos de Instalación

1. Instala la extensión del navegador que coincida con tu navegador web desde la tienda web oficial o los enlaces en el modal de descarga.
2. Asegúrate de que la aplicación de escritorio AB Download Manager esté ejecutándose en segundo plano.
3. Una vez activa, los enlaces que hagas clic en el navegador se enrutarán automáticamente directamente a AB Download Manager.
