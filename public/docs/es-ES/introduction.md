# Introducción

¡Bienvenido a la documentación de **AB Download Manager**! 

AB Download Manager es un gestor de descargas gratuito, rápido y de código abierto diseñado para hacer que las descargas de archivos sean fluidas, organizadas y extraordinariamente rápidas. 

## Características Principales

- **Descargas Rápidas**: Acelera tus descargas hasta un 500% usando conexiones multi-hilo.
- **Gestión de Colas**: Descarga múltiples archivos secuencialmente creando colas de descarga personalizadas.
- **Interfaz Elegante**: Disfruta de una interfaz de usuario hermosa y moderna con soporte integrado para modo oscuro.
- **Integración del Navegador**: Captura automáticamente enlaces de navegadores populares (Chrome, Firefox, Edge, Opera, etc.) usando extensiones.
- **Limitador de Velocidad**: Controla cuáncho ancho de banda consume la aplicación para evitar ralentizar tu red.
